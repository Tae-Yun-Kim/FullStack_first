     - React

     - jwt
     - Spring JPA
     - MySQL 8.0

## 주요 기능
* 밀키트 구매 시 재료의 수량 가감
* 밀키트 이름 및 재료의 이름으로 검색 가능
* 게시판을 이용한 회원간의 레시피 공유
* Python tensorflow를 이용한 이미지 검색
